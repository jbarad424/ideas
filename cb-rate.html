The Tinder-style swipe interface has been replaced with a unified Gallery inside the main review page. Everything you could do here you can now do there, with better support for shared team state:

  • See all images (rated + unrated) in the Gallery grid
  • Show rejects toggle in the Gallery filter bar
  • Super-like via the gold ⭐ button in each image's overlay
  • Score any image by clicking the score badge
  • Download full-resolution, send to Video Lab, or generate variations
  • Shared across all team members via cb-feedback.json sync
