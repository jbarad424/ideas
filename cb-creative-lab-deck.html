The Problem

Expensive Shoots

Professional product photography runs $500-$2,000 per shoot. For a startup iterating on designs, that adds up fast.

Wrong

AI Inconsistency

Off-the-shelf AI generation produces inconsistent, wrong-looking products. Wrong button placement, wrong band shape, wrong everything.

Zero

No Feedback Loop

Every generation starts from scratch. The AI never learns what your product actually looks like or what your brand needs.

Waste

Wasted Cycles

Teams spend hours reviewing assets that don't match the brand. Time that should go toward building and selling.

The Solution

A creative pipeline that
actually learns

The Journey

11 versions in one day

From zero to a learning creative pipeline in a single afternoon.

v1 – v2 · Foundation

Three engines, one UI, 21 assets

Built FLUX, Recraft, and Runway integrations. Swipe-based rating interface. First batch of 21 generated assets.

v3 · Accessibility

Bigger text, positive feedback

Team said: text too small, and we should capture what works, not just what's wrong. Added positive feedback popups.

v4 · Identity

Per-user identity + prompt refinement

Each team member gets their own identity. Generate More triggers real prompt refinement based on prior ratings.

v5 · Intelligence

Smart directions + multi-format batches

User direction nudges, balanced product + action shot mix, Runway video included in every batch.

v6 · Research-Backed

5-layer prompt engine

Frozen product blocks, positive inversion of negative feedback, preservation language to protect what already works.

v7 – v8 · Polish

Brand identity + evolution dashboard

Brand name engines, team passcodes, and a clean dashboard showing system-wide learning progress.

v9 · AI Co-Rater

Claude Vision enters the loop

Every generated image is now analyzed by Claude Vision. Product accuracy, scene quality, and brand alignment scored automatically.

v10 · Reliability

Error handling + progress UI

Fixed Recraft API edge cases. Added real-time generation progress so the team can watch assets being created.

v11 · Full Loop

Written feedback becomes prompt instructions

Natural language feedback auto-translates to generation guidance. The loop is complete.
